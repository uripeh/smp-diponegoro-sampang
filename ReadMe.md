# 🏫 SMP Diponegoro Sampang

![Banner Sekolah](https://capsule-render.vercel.app/api?type=waving&color=1abc9c&height=200&section=header&text=SMP%20Diponegoro%20Sampang&fontSize=40&fontColor=fff)

Selamat datang di repositori resmi **SMP Diponegoro Sampang**. Repositori ini digunakan sebagai pusat pengembangan digital, pengelolaan bahan ajar, serta dokumentasi projek teknologi dan informasi sekolah.

---

## 📌 Profil Singkat
SMP Diponegoro Sampang berkomitmen untuk mengintegrasikan nilai-nilai pendidikan karakter dengan pemanfaatan teknologi informasi guna mencetak generasi yang cerdas, kreatif, dan adaptif terhadap perkembangan zaman.

* **Lokasi:** Sampang, Cilacap, Jawa Tengah, Indonesia
* **Fokus Digital:** Sistem Informasi Akademik.

---

## 📂 Struktur Repositori
Berikut adalah peta folder (direktori) yang digunakan untuk mengelola sistem informasi dan administrasi digital di SMP Diponegoro Sampang:

```text
📂 smp-diponegoro-sampang
 ├── 📁 profil-sekolah/       # Visi, misi, sejarah, dan fasilitas sekolah
 ├── 📁 jadwal-kelas/         # Jadwal pelajaran, kalender akademik, dan info piket
 ├── 📁 guru-wali/            # Data wali kelas, pembagian tugas, dan catatan kelas
 ├── 📁 e-rapor-akademik/     # Pengolahan nilai, capaian belajar, dan arsip rapor
 ├── 📁 e-presensi/           # Sistem rekap absensi guru, staf, dan siswa
 ├── 📁 e-perpustakaan/       # Katalog buku digital, sirkulasi peminjaman, dan literasi
 ├── 📁 administrasi-spp/     # Pencatatan keuangan, SPP, dan infaq pembangunan
 └── 📁 layanan-konseling/    # Dokumentasi bimbingan konseling (BK) & perkembangan siswa
