Konfigurasi API Google Sheets

Tempelkan URL Google Apps Script Web App (API) Anda untuk menghubungkan formulir login sekolah secara dinamis dengan database spreadsheet.

Panduan Sinkronisasi Database

Mengapa Sinkronisasi Terhambat?

Biasanya browser memblokir koneksi API (*CORS*) karena Apps Script dideploy secara privat atau meminta hak akses otentikasi akun Google Anda pribadi.

Solusi Singkat Deploy:

  1. Buka Google Spreadsheet > **Ekstensi** > **Apps Script**.
  2. Klik **Terapkan (Deploy)** di kanan atas > **Kelola Penerapan**.
  3. Klik ikon pensil edit pada versi Anda, ubah hak akses "Who has access" ke "Anyone" (Siapa Saja).
  4. Deploy ulang, salin URL baru dan simpan di menu pengaturan gerigi web ini.
